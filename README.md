## 📂 Repository Structure

The project is organized into modular categories, making it easy to navigate through specific algorithms and themes.

### 🏛️ [Machine Learning](./Machine-Learning/)
Focused on classical supervised and unsupervised learning techniques.

> [!TIP]
> Each subfolder contains a **`REVISION_GUIDE.md`** folder with summarized explanations and visual infographics for quick review!

*   **[Supervised Learning](./Machine-Learning/Supervised/)**
    *   **[Classification](./Machine-Learning/Supervised/Classification/)**: Random Forest, SVM, Decision Trees, KNN, and Logistic Regression.
    *   **[Regression](./Machine-Learning/Supervised/Regression/)**: Linear and Logistic regression models, plus combined exploratory analysis.
    *   **[Projects](./Machine-Learning/Supervised/Projects/)**: End-to-end applications like the [Placement Chance Predictor](./Machine-Learning/Supervised/Projects/Placement_Chance_Predictor/).
*   **[Unsupervised Learning](./Machine-Learning/Unsupervised/)**
    *   **[Clustering](./Machine-Learning/Unsupervised/Clustering/)**: K-Means (scratch & sklearn), Agglomerative & Divisive hierarchical clustering.
    *   **[Dimensionality Reduction](./Machine-Learning/Unsupervised/Dimensionality_Reduction/)**: PCA analysis for feature optimization.

### ⚡ [Deep Learning](./Deep-Learning/)
Implementations of neural networks and deep-learning concepts.

*   **[Basics](./Deep-Learning/Basics/)**: Perceptron implementations from scratch.
*   **[MLP](./Deep-Learning/MLP/)**: Multilayer Perceptrons for complex tasks like MNIST and Customer Churn prediction.
*   **[PyTorch](./Deep-Learning/PyTorch/)**: Focused studies on PyTorch Tensors, Autograd, and Simple Neural Networks.

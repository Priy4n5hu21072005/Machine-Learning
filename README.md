# AI & Machine Learning Repository

Welcome to my comprehensive collection of Artificial Intelligence and Machine Learning implementations. This repository contains various algorithms, projects, and practice notebooks ranging from foundational Machine Learning to advanced Deep Learning techniques.

## 🚀 Repository Structure

The repository is organized into two main domains:

### 1. [Machine Learning](./Machine-Learning/)
This section covers classic Supervised and Unsupervised Learning algorithms implemented from scratch and using popular libraries like Scikit-Learn.

- **Supervised Learning**: 
  - **Projects**: End-to-end applications like the [Placement Chance Predictor](./Machine-Learning/Supervised-Learning/Projects/Placement_Chance_Predictor/).
  - **Algorithms**: Implementation of Linear Regression, Logistic Regression, Random Forest, Decision Trees, KNN, and SVM.
- **Unsupervised Learning**:
  - **Algorithms**: K-Means Clustering (Scratch & Scikit-Learn), Principal Component Analysis (PCA), and Hierarchical Clustering.

### 2. [Deep Learning](./Deep-Learning/)
Focused on Neural Networks and advanced architectures.

- **Perceptron**: Logical gates and basic perceptron implementations from scratch.
- **Multilayer Perceptron (MLP)**: Complex neural networks for tasks like Customer Churn prediction using TensorFlow/Keras.

## 🛠️ Tech Stack

- **Languages**: Python
- **Libraries**: NumPy, Pandas, Scikit-Learn, TensorFlow, Keras, Matplotlib, Seaborn
- **Tools**: Jupyter Notebooks, FastAPI (for project deployment)

## 📁 Projects Highlights

### [Placement Chance Predictor](./Machine-Learning/Supervised-Learning/Projects/Placement_Chance_Predictor/)
A full-stack ML application that predicts student placement chances based on various parameters. Includes EDA, preprocessing, model training, and a FastAPI deployment.

## 📝 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Priy4n5hu21072005/Machine-Learning.git
   ```
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Explore the notebooks**: Navigate to any folder and open `.ipynb` files in Jupyter or VS Code to see the implementations.
